<p> Amarok是一款Android應用，只需一鍵即可隱藏你的私人文件和應用。</p><p> 我們常常使用複雜的加密工具來保護文件和應用，目的是保護它們免受他人窺視。這些加密工具雖然有效，但通常速度緩慢且資源密集，使得加密視頻和音樂等大文件變得艱鉅。儘管這些方法安全性強，但對於普通用戶來說，這些方法往往是過於大材小用，因爲他們只是想將私人文件和應用隱藏在無意中。</p><p> Amarok被設計成輕量級隱藏器。它掩蓋了文件名和頭部，使其看起來“損壞”且無法打開。Amarok 還會禁用應用程序，使其在啓動器和系統菜單中不可見。</p><p><b>特色</b></p><ul> <li><strong>用戶友好</strong>: 只要一鍵即可隱藏應用程序或文件</li> <li><strong>Rapid and Efficient Large File Hiding</strong>: Hides by altering only file name and signature, unaffected by file size.</li> <li><strong>Root-Free Application Hiding</strong>: Makes apps vanish from the launcher. Compatible with Root, Shizuku, Dhizuku, and DSM modes.</li> <li><strong>Panic button</strong>: Use a floating button to quickly hide applications and files in urgent scenarios. </li> <li><strong>Quick Settings Tile</strong>: A control center toggle for immediate hiding, bypassing the need to launch the app.</li> <li><strong>App Lock</strong>: Secure Amarok access with a password or fingerprint.</li> <li><strong>Pleasant Interface</strong>: Clean Material3 design.</li></ul><p>Please be aware that Amarok is <b>not an encryption software</b>, but rather a tool for hiding things. <b>We strongly advise against using Amarok to protect confidential files and applications.</b></p>
